# CaloFitPro
A Small Android App to compute your calories

## Activité 3
Pour l'activité 3, j'ai créé un bouton sur la page principale: `VOIR MON GRAPHIQUE`. Ce Bouton permet d'accéder à la 3ème activitée.

Elle Permet de voir tout les calculs fait précédement ainsi que la date ou le calcul à été fait. Un dernier bouton: `SUPPRIMER LE DERNIER` permet de supprimer le dernier calcul à chaque fois qu'on appuie dessus.
